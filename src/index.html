ユーザー管理・権限設定・案内メール送信

使い方ガイド

1. ユーザーを追加する

「ユーザー追加」をクリックし、メールアドレスを入力して「作成」。user_メールの@の前 形式でLinuxユーザーが自動作成され、SSH鍵も自動生成されます。

2. 一括で追加する

「一括追加」をクリックし、メールアドレスを1行ずつ入力。全員分をまとめて作成できます。

3. 案内メールを送る

ユーザー一覧の「メール送信」をクリックすると、メール内容がプレビューされます。確認後「送信」で案内メールが届きます。メールにはSSH秘密鍵の添付と、Gemini用の接続ガイドプロンプトが含まれます。

4. 権限の仕組み

sandbox = 全員が読み書き可能。app1/app2 = 担当者のみ。sudo/docker = 一般ユーザーには付与されません。

メール送信について

案内メールにはGemini用プロンプトが含まれており、受信者はそのプロンプトをGeminiに貼り付けるだけで、完全初心者でもSSH接続の設定を一歩ずつ進められます。

SMTP設定が必要です。docker-compose.yml の SMTP_USER と SMTP_PASS を設定してください（Gmailの場合はアプリパスワード）。

⚠ SMTP（メール送信）が設定されていません。メールのプレビューは可能ですが、送信にはdocker-compose.ymlでSMTP設定が必要です。
